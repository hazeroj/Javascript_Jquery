제이쿼리 문서객체조작-내용추가_삭제메서드

DOM(문서객체모델)에서 문서안의 요소에 내용을 추가하거나 삭제를 할 수 있다.

  • append(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 뒤쪽에 내용을 추가
  • prepend(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 앞쪽에 내용을 추가
  • after(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨뒤에 내용을 추가
  • before(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨앞에 내용을 추가

insertAfter(), insertBefore()를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고, B라는 요소의 뒤쪽에 A요소를 배치할 수도 있다. after(), before()메소드와 기능은 같지만 사용법이 다르다.

예)
$('A').after('B'); - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.
$('B').insertAfter('A'); - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.

아래 remove버튼을 클릭하면 이 박스요소가 제거됩니다.
아래 empty버튼을 클릭하면 이 박스요소안에 있는 내용만 제거됩니다.
